Pour résoudre cela, voici quelque petites aides qui peux vous aider !

  • Changement d'adresse IP (dns)
    Il se peut que vous avez modifier vos DNS recemment, cela dit qu'il faut une attente d'environs 1h à 12h selon certain hébergeur. Pour cela, attendez que la propagation soit fini. Sinon, rendez-vous sur l'aide cPanel et suivez les instructions pour supprimer les caches
  • Erreur lors du transfère
    Vous devez vérifier que votre fournisseur d'hébergement dispose de l'adresse IP correcte configurée pour vos paramètres Apache et vos enregistrements DNS. Un redémarrage d'Apache peut être nécessaire pour que les nouveaux paramètres prennent effet.
  • Changement de serveur
    L'URL de ce domaine peut être changé par le fournisseur d'hébergement ou peut être déplacé vers un serveur différent
